国内大家如果想要注册和使用 ChatGPT，需要满足 2 个前提条件：

1. **可以科学上网**
2. **有 OpenAI 支持注册地区的海外手机号**

如果想要更稳定快速的使用 ChatGPT，那么建议申请自己的 API Key 后使用第三方客户端，虽然现在新用户只有 5 美元的额度，但也非常够用了。

下面是我自己目前觉得比较好的注册和使用方式供大家参考

## 1. 注册 OpenAI 账号

1. 搜索公众号 **易博通香港 eSender HK** ，关注后点击下方“订购”— “订购英国手机号”，然后在新页面购买一个英国手机号（支持支付宝/微信），后面手机号收到的信息都会通过公众号消息推送给你：

![](https://imagedelivery.net/8B08sdLvw783CQcaKhUoYw/24a50e2c-6796-403a-98a4-f1416db58400/public)

2. 前往 https://chat.openai.com， 点击 Sign up 注册账号，注册时可以使用邮箱，也可以直接用谷歌或者微软账号注册：

![](https://imagedelivery.net/8B08sdLvw783CQcaKhUoYw/3299977a-73bc-40de-020d-64971f42e100/public)

3. 然后会验证手机号，这时候选择 **United Kingdom**，使用刚才购买的手机号接受验证码，帐号注册成功.

## 2. 使用第三方客户端

因为直接在官网使用速度会比较慢，推荐通过申请自己的 API Key 然后使用第三方客户端，下面以 ChatKit 为例：

1. 打开个人 OpenAI API Keys 管理界面：https://platform.openai.com/account/api-keys

![](https://imagedelivery.net/8B08sdLvw783CQcaKhUoYw/131664ce-09f7-436c-d1c4-9f38c9482700/public)

2. 点击 “Create new secret key”，创建一个新的 Key：

![](https://imagedelivery.net/8B08sdLvw783CQcaKhUoYw/8c5347fc-2ee9-4965-d465-259fe99bf500/public)

3. 不要关掉这个页面和窗口，在开一个新的标签页，打开 ChatKit：https://chatkit.app/

![](https://imagedelivery.net/8B08sdLvw783CQcaKhUoYw/5a138cc3-b5a4-49d4-6bdd-289cbda4ed00/public)

4. 回到刚才生成 Key 的窗口，点击绿色的复制按钮将 Key 复制到剪贴板，粘贴到 ChatKit 中的“OpenAI API Key” 输入框中，点击 “Save”：

![](https://imagedelivery.net/8B08sdLvw783CQcaKhUoYw/7c7219a2-8300-48c6-a48b-f4b84e611200/public)

5. 然后就可以正常使用了，速度和稳定性都比在官网使用更好，而且 ChatKit 每次问答还有使用提示，用起来很舒服：

![](https://imagedelivery.net/8B08sdLvw783CQcaKhUoYw/26d70d2e-bc48-407f-26e9-ac780e396700/public)
